# Creswick Soccer Club Website - Sprint 2 Final V3

This is the organised Version 3 front-end website for the Creswick Soccer Club ITECH3208 project.

## Project status

This version is a static front-end prototype built with HTML, CSS and JavaScript.

Included Sprint 2 work:
- final V3 visual design
- responsive navigation and mobile hamburger menu
- home hero image slideshow
- club values banner
- Leagues pages for Seniors, Juniors and MiniRoos
- Gallery page with 3D hover interaction
- Contact page with client-side validation
- Admin Login UI hidden from the main public navigation
- cleaned folder structure for GitHub version control

Backend authentication, email sending, database storage and live data integration are planned for a later sprint/semester.
