# Creswick Soccer Club Website

This is Version 1.0 of the Creswick Soccer Club website project for ITECH 3208.

## Project Description

The website is a static HTML, CSS, and JavaScript prototype for Creswick Soccer Club.

It includes pages for:

- Home
- Teams
- Sponsors
- Registrations and Kits
- Fixtures and Results
- Policies and Codes of Conduct
- Contact
- Admin Login

## Version

Version 1.0

## Team

Built by ITECH 3208 Team, Federation University.

## Current Status

This version is a static front-end prototype. Backend features such as form submission, admin authentication, database connection, and full admin functionality are planned for a future version.

## How to Run

Open `index.html` in a web browser.

## Notes

- This project currently uses a single HTML file containing the HTML, CSS, and JavaScript.
- The website is designed as a front-end prototype for the first version submission.
- Future versions can separate CSS, JavaScript, and image assets into separate folders.
